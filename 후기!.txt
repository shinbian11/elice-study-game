[박세령 - 테트리스]
<소개>
- 테트리스는 누구나 살면서 한번쯤 해보았을 게임입니다.
- 혹시 사용법이 궁금하시다면 우측 상단의 물음표 아이콘을 클릭해보세요!

- 초기에 테트리스 블럭의 모양과 색을 설정해주었습니다.
- 크게 키 입력처리, 도형 생성, 도형 조작, 레벨/점수 판정, 게임종료/멈춤으로 이루어져있습니다.
- random 함수를 이용해 도형이 내려옴과 동시에 다음 도형을 보여주도록 했습니다.
- 도형이 1줄이 꽉 찼을 때 그 줄이 없어지고 위에 블럭이 내려오며 점수가 올라갑니다. (checkLine(), iSFull(), removeLine())
- 도형을 없애 점수가 올라갈수록 레벨이 올라가고, 도형이 내려오는 속도가 빨라집니다.

<소감>
- 기획부터, 디자인, 개발까지 게임 사이트를 처음부터 만든 다는 것 자체가 도전이었지만, 팀원들과 다같이 힘을 내서 좋은 결과를 얻은 것 같습니다!
- 매 회의 때마다 팀원들간의 피드백 시간, 칭찬의 시간이 너무 뜻깊었습니다.
- 서로가 잘 아는 분야를 공유하며 배우고 성장하는 프로젝트였습니다.
- 이런 기회를 가질 수 있게 해준 엘리스 고마워요 :) ♥.  

[김지환 - 룰렛]
<소개>
- 룰렛은 간단한 규칙을 가지고 있습니다. start 버튼을 눌러 룰렛을 돌리고, 룰렛이 멈추는 위치에 따라 다양한 상품을 받을 수 있습니다.
- 룰렛의 자바스크립트 코드는 Roulette이라는 클래스에 모두 만들었습니다.
- 그 클래스 내부는 크게 5가지의 메소드로 구성되어 있습니다. 이 메소드들은 start 버튼을 클릭하여 이벤트가 발생했을 때 실행되는 event handler 들입니다.

- rotateRoulette 메서드를 통해 룰렛을 회전시킵니다. 
- displayResult 메서드를 이용하여 룰렛 결과 메세지를 보여줍니다.
- 결과를 보여준 이후, 다시하기 버튼을 눌렀을 때 resetRoulette 메서드와 enabledRoulette 메서드가 차례로 실행됩니다.
- resetRoulette 메서드는 시작 직전의 위치로 룰렛을 위치시킵니다. (초기화)
- enabledRoulette 메서드는 룰렛이 회전되는 동안 disabled 되었던 start 버튼을 재활성화시켜, 룰렛을 다시 즐길 수 있도록 합니다.

<소감>
- 엘리스 교육과정에서 배웠던 부분들을 최대한 모두 적용하려고 노력하였습니다.
- Javascript 파일에서는, 비동기 함수와 async/await를 이용하여 메서드를 구현하였고, class 내부에 이 메서드들을 구현하며 재사용성의 필요성에 대해서도 배울 수 있었습니다.
- 또한, html, css의 프론트엔드 부분을 수정하고 개선하면서 UI 디자인적인 부분도 쉽지 않구나라는 점을 느꼈습니다.
- 협업을 하는 중간중간 디스코드 채널과 엘리스 회의 공간을 통해 토론하고, 깃허브로 커밋 관리를 하면서 협업하는 방법을 배웠습니다.
- 시간적인 여유가 많지 않아, 작은 화면이나 모바일 창에서의 반응형 처리에 아쉬움이 많은데, 프로젝트 기간이 끝나더라도 개인적인 시간을 내어서 완성할 예정입니다.
